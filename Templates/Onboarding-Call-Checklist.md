# Onboarding Call Checklist

## Before the call
- [ ] Review intake form responses
- [ ] Run Stage 1 Pre-Call Analysis
- [ ] Note any red flags (medical conditions, ED screening, GLP-1)
- [ ] Prepare questions about gaps in intake data

## During the call (15-30 min)
- [ ] Introduce yourself and set expectations for the call
- [ ] Confirm their primary goal and timeline
- [ ] Ask about training history and current routine in detail
- [ ] Ask about nutrition: what does a typical day of eating look like?
- [ ] Ask about sleep quality and stress levels
- [ ] Ask about past attempts: what have you tried and why did it stop working?
- [ ] Ask about schedule: work hours, gym availability, meal prep time
- [ ] Ask about injuries or pain (current and past)
- [ ] Ask about supplements they currently take
- [ ] Discuss their preferred communication style
- [ ] Set expectations: blueprint in 3-5 days, biweekly check-ins, iMessage access
- [ ] Ask if they have any questions

## After the call
- [ ] Write call notes in detail
- [ ] Run Stage 1.5 Call Notes Integration
- [ ] Run Calc Layer
- [ ] Run Stage 2 Blueprint Generation
- [ ] Run cleanup-text.js and verify-numbers.js
- [ ] Run Stage 3 QA Gate
- [ ] Review blueprint personally
- [ ] Export to PDF
- [ ] Send via iMessage with a personal message
